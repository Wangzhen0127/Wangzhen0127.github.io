关于和GitHub和vscode斗智斗勇

2022年12月1日 · 阅读需 1 分钟

配置deploy的时候出现了

[ERROR] Deployment of the build output failed .
[ERROR] Error: Running" git push" command failed. Does the GitHub user account you are using have push access to the repository?

这种问题，在输入的时候明明提交了密码 不知道为什么输入不上去 重复的更新了vscode的账号密码还是不行

发现了不能输入密码 得输入token才可以 这边输入了两次 不能再次输入密码了 搜索了一下 需要用下面的指令

git config --system --unset credential.helper

就可以在git push的时候输入密码了
