Set

概念​

很像加强版的Map

基本API​

new 关键字和 Set 构造函数可以创建一个空集合​

const m = new Set();

基本操作​

add()增加值

has()查询

size 取得元素数量

delete()和 clear()删除元素

顺序和迭代​

可以直接对集合实例使用扩展操作，把集合转换为数组

const s = new Set(["val1", "val2", "val3"]); console.log([...s]); // ["val1", "val2", "val3"]

回调方式​

const s = new Set(["val1", "val2", "val3"]); s.forEach((val, dupVal) => console.log(`${val} -> ${dupVal}`)); 
// val1 -> val1 // val2 -> val2 // val3 -> val3

Map

复习​

ES6之前 用object存储键值对

对象属性作为键，再使用属性来引用值

概念​

Map 是一种新的集合类型，为这门语言带来了真正的键/值存储机制。Map 的大多数特性都可以通过 Object 类型实现

基本api​

new 关键字和 Map 构造函数可以创建一个空映射​

创建的同时初始化实例，可以给 Map 构造函数传入一个可迭代对象，需要包含键/值对数组

使用嵌套函数

const m1 = new Map(
    [ ["key1", "val1"], 
     ["key2", "val2"],  
     ["key3", "val3"]])
 console.log(m1.size); // 3

使用自定义迭代器

const m2 = new Map({  
[Symbol.iterator]: function*() {  
yield ["key1", "val1"];  
yield ["key2", "val2"];  
yield ["key3", "val3"];  } }); 
console.log(m2.size);
set()方法添加键/值对​

set()方法返回映射实例，因此可以把多个操作连缀起来，包括初始化声明：

const m = new Map().set("key1", "val1"); m.set("key2", "val2")  
 .set("key3", "val3"); 
console.log(m.size);
get()和 has()查询​

size 属性获取映射中的键/值对的数量

使用 delete()和 clear()删除值​

与object进行对比​

Object 只能使用数值、字符串或符号作为键

Map 可以使用任何 JavaScript 数据类型作为键

选择Object还是Map​

内存占用​

Map 大约可以比 Object 多存储 50%的键/值对

插入性能​

Map 的性能更佳

查找速度​

只包含少量键/值对，则 Object 有时候速度更快

把 Object 当成数组使用，内存中使用更高效的布局

如果代码涉及大量查找操作，选择 Object 更好

删除性能​

Map 的 delete()操作都比插入和查找更快。如果代码涉及大量删除操作，选择 Map
